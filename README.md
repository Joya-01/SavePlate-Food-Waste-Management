**SavePlate** is a full-stack web application designed to reduce food wastage by connecting donors (hotels, restaurants) with NGOs and charities. This project is a core part of my final year B.Tech CSE portfolio.

## üöÄ Project Overview
SavePlate creates an efficient bridge for leftover food through three key modules:
* **User (Donors):** Register and list excess food with quantity and type.
* **Admin (NGOs):** Verify and manage donations for distribution.
* **Delivery:** Coordinate pickup and drop-off logistics.

## üìÇ Setup Instructions
1. **Clone & Move:** Place the project folder in your XAMPP `htdocs` directory.
2. **Start Server:** Open XAMPP Control Panel and start **Apache** and **MySQL**.
3. **Database Setup:** - Open `http://localhost/phpmyadmin`.
   - Create a database named `food-waste-management`.
   - Import the `demo.sql` file from the `/database` folder.
4. **Run:** Access the app at `http://localhost/SavePlate-Food-Waste-Management/`.
